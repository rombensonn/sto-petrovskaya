# СТО на Петровской, 52

Индустриальный одностраничный лендинг автосервиса в деревне Шишкино.

## Что внутри

- `index.php` — основная PHP-версия с обработкой заявки и сохранением лидов в `data/leads.php`.
- `index.html` — статическая версия для GitHub Pages.
- `assets/` — стили, JS и тематические изображения.

## Важно про GitHub Pages

GitHub Pages не выполняет PHP. Поэтому публичная Pages-ссылка открывает статическую версию сайта. Для рабочей отправки формы используйте `index.php` на хостинге с PHP.
